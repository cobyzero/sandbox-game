# sandbox-game
Sandbox game basic

Game basic inspired by Roblox.

## Features
- Place and remove blocks
- Save and load worlds

## Controls
- Left Click: Place Block
- Right Click: Remove Block
- W/A/S/D: Move
- Space: Jump
- Shift: Sprint

## Engine
Made with [Godot Engine](https://godotengine.org/) 4.0
